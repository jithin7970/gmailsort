Connect your real Gmail (optional)

This page runs entirely in your browser — no server, nothing stored remotely. To pull your actual inbox, it talks directly to Google's Gmail API using your own OAuth client. Right now it's showing a sample inbox so you can try the sorting first.

How do I get a Client ID? (2 minutes, one-time)

1. Go to console.cloud.google.com → create a project (or pick one).
2. Enable the Gmail API under "APIs & Services".
3. Go to "Credentials" → "Create credentials" → "OAuth client ID" → type Web application.
4. Under "Authorized JavaScript origins" add the URL this page is running on.
5. Copy the Client ID and paste it above. It's saved only in your browser (localStorage), never sent anywhere except Google.
Scope used: gmail.readonly + gmail.modify (needed to archive/trash/label from this page).
